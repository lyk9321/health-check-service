## 4.2 기능별 상세 명세

### F-02: 신체정보 입력 및 BMI 계산

**입력**:
- 키: 숫자 (cm), 유효 범위 100~250
- 몸무게: 숫자 (kg), 유효 범위 20~300

**처리**:
```python
def calculate_bmi(height_cm: float, weight_kg: float) -> float:
    height_m = height_cm / 100
    return round(weight_kg / (height_m ** 2), 1)
```

**출력**:
- BMI 수치 (소수점 1자리)
- BMI 분류 문구 (저체중/정상/과체중/비만)

### F-04: 모름 처리

**미입력 항목별 처리 방식**:

| 미입력 항목 | 처리 방식 | 화면 문구 |
|-----------|----------|----------|
| 혈압 | 해당 성별·연령대 평균값으로 대체 | 혈압 미입력으로 혈압관리 평가는 제한됩니다. |
| 공복혈당 | 해당 성별·연령대 평균값으로 대체 | 공복혈당을 입력하면 혈당관리 결과가 더 구체화됩니다. |
| 지질 수치 (4종 모두 미입력) | 지질 관련 룰 기반 해석 생략 | 이상지질혈증 검사 수치(LDL/HDL/중성지방/총콜레스테롤)를 입력하면 지질관리 안내를 받을 수 있습니다. |
| 지질 수치 (일부만 입력) | 입력된 항목만 해석, 미입력 항목 생략 | (입력된 항목만 안내 표시) |
| 생활습관 | 추천 문구 일부 생략 | 생활습관 정보를 추가하면 추천 내용이 더 구체화됩니다. |
| 운동 정보 | 운동 추천 상세화 제외 | 운동 정보를 입력하면 신체활동 관련 추천이 더 구체화됩니다. |

### F-06: 입력 완성도 계산

**완성도 가중치**:

| 항목 | 가중치 | 비고 |
|------|--------|------|
| 성별, 연령대, 키, 몸무게 | 35% (필수) | 미입력 시 서비스 이용 불가 |
| 생활습관 (흡연, 음주) | 15% | |
| 혈압 | 15% | |
| 공복혈당 | 15% | |
| 지질 수치 | 10% | LDL·HDL·TG·총콜레스테롤 중 **1개 이상** 입력 시 달성 |
| 허리둘레 | 5% | |
| 운동 정보 | 5% | |

```python
def calculate_completeness(user_input: dict) -> int:
    """입력 완성도를 0~100% 정수로 반환"""
    score = 0

    # 필수 (35%)
    if all(user_input.get(k) for k in ['gender', 'age_group', 'height_cm', 'weight_kg']):
        score += 35

    # 생활습관 (15%)
    if user_input.get('lifestyle'):
        score += 15

    # 혈압 (15%)
    if user_input.get('blood_pressure'):
        score += 15

    # 공복혈당 (15%)
    if user_input.get('fasting_glucose'):
        score += 15

    # 지질 수치 (10%) — 4종 중 1개 이상 입력 시 달성
    lipid_keys = ['ldl_cholesterol', 'hdl_cholesterol', 'triglyceride', 'total_cholesterol']
    if any(user_input.get(k) for k in lipid_keys):
        score += 10

    # 허리둘레 (5%)
    if user_input.get('waist'):
        score += 5

    # 운동 정보 (5%)
    if user_input.get('exercise'):
        score += 5

    return score
```

### F-07: 룰 기반 건강지표 해석

**BMI 판정 기준**:

| 범위 | 분류 | 출력 문구 |
|------|------|----------|
| < 18.5 | 저체중 | 체중 증가 관련 관리가 필요할 수 있습니다. |
| 18.5 ~ 22.9 | 정상 | 체중 범위는 현재 적정 수준입니다. |
| 23.0 ~ 24.9 | 과체중 | 체중 관리를 점검해볼 필요가 있습니다. |
| ≥ 25.0 | 비만 | 체중관리가 도움이 될 수 있습니다. |

**혈압 판정 기준**:

| 범위 (수축기/이완기) | 분류 | 출력 문구 |
|-------------------|------|----------|
| < 120 / < 80 | 정상 | 현재 혈압 범위는 적정 수준입니다. |
| 120~129 / < 80 | 주의 | 혈압이 높아지는 경향이 있어 관리를 권장합니다. |
| 130~139 / 80~89 | 고혈압 전단계 | 혈압 재측정 및 생활습관 관리를 권장합니다. |
| ≥ 140 / ≥ 90 | 고혈압 범위 | 혈압 재측정 및 의료기관 상담을 권장합니다. |

**공복혈당 판정 기준**:

| 범위 | 분류 | 출력 문구 |
|------|------|----------|
| < 100 | 정상 | 현재 공복혈당 범위는 적정 수준입니다. |
| 100~125 | 공복혈당 장애 | 공복혈당 관리가 필요할 수 있습니다. |
| ≥ 126 | 당뇨병 범위 | 가까운 의료기관에서 확인검사를 권장합니다. |

**지질 수치 판정 기준** (한국지질·동맥경화학회 2022):

> ⚠️ **콜레스테롤은 4종 각각 입력 가능. 입력된 항목만 해석하며, LDL을 최우선 표시.**  
> 이상지질혈증 검사를 받지 않은 경우 4종 모두 미입력이 정상. 총콜레스테롤만 아는 경우 보조 참고로 표시.

**① LDL 콜레스테롤** — 심혈관 위험 1차 지표. 낮을수록 좋음.

| 범위 (mg/dL) | 분류 | 출력 문구 |
|-------------|------|----------|
| < 100 | 적정 | LDL 콜레스테롤은 적정 범위에 해당합니다. |
| 100~129 | 정상 상한 | LDL 콜레스테롤은 정상 범위에 해당합니다. |
| 130~159 | 경계 | LDL 콜레스테롤이 경계 범위에 해당하므로 관리 점검이 도움이 될 수 있습니다. |
| 160~189 | 높음 | LDL 콜레스테롤이 높은 범위에 해당하므로 의료기관 상담을 권장합니다. |
| ≥ 190 | 매우 높음 | LDL 콜레스테롤이 매우 높은 범위에 해당하므로 의료기관 상담을 권장합니다. |

**② HDL 콜레스테롤** — 높을수록 좋음(심혈관 보호 인자). 성별에 따라 기준 상이.

| 범위 (mg/dL) | 성별 | 분류 | 출력 문구 |
|-------------|------|------|----------|
| < 40 | 남성 | 낮음 | HDL 콜레스테롤이 낮은 범위에 해당합니다. 규칙적인 신체활동이 도움이 될 수 있습니다. |
| < 50 | 여성 | 낮음 | HDL 콜레스테롤이 낮은 범위에 해당합니다. 규칙적인 신체활동이 도움이 될 수 있습니다. |
| 40~59 (남) / 50~59 (여) | 공통 | 보통 | HDL 콜레스테롤은 보통 범위에 해당합니다. |
| ≥ 60 | 공통 | 높음 | HDL 콜레스테롤이 높은 범위에 해당합니다. |

**③ 트리글리세라이드(중성지방)** — 낮을수록 좋음.

| 범위 (mg/dL) | 분류 | 출력 문구 |
|-------------|------|----------|
| < 150 | 적정 | 중성지방은 적정 범위에 해당합니다. |
| 150~199 | 경계 | 중성지방이 경계 범위에 해당하므로 관리 점검이 도움이 될 수 있습니다. |
| 200~499 | 높음 | 중성지방이 높은 범위에 해당하므로 의료기관 상담을 권장합니다. |
| ≥ 500 | 매우 높음 | 중성지방이 매우 높은 범위에 해당하므로 의료기관 상담을 권장합니다. |

**④ 총콜레스테롤** — LDL/HDL/TG 미입력 시 보조 참고 지표로만 사용.

| 범위 (mg/dL) | 분류 | 출력 문구 |
|-------------|------|----------|
| < 200 | 적정 | 총콜레스테롤은 적정 범위에 해당합니다. |
| 200~239 | 경계 | 총콜레스테롤이 경계 범위에 해당하므로 관리 점검이 도움이 될 수 있습니다. |
| ≥ 240 | 높음 | 총콜레스테롤이 높은 범위에 해당하므로 지질 검사 결과 확인을 권장합니다. |

> LDL·HDL·TG 중 1개 이상 입력된 경우 총콜레스테롤 해석은 표시하지 않음(중복 혼란 방지).

### F-09: 비교군 비교

**비교 항목**:
- BMI, 수축기혈압, 공복혈당, 허리둘레 (입력된 항목만)
- 비교 방식: 사용자 수치 vs. 비교군 평균 ± 표준편차

> **지질 수치는 비교군 비교 제외**: 건강검진 데이터에서 콜레스테롤 항목이 ~66% 결측이므로  
> 비교군 통계의 신뢰도가 낮음. 지질 수치는 룰 기반 해석으로만 안내.

**출력 형태**:
```
[비교군: 30대 여성 검진자 그룹 (n=12,450명)]
• BMI: 당신 23.5 | 비교군 평균 22.8 (±3.1) → 비슷한 수준
• 수축기혈압: 당신 128 | 비교군 평균 116 (±13.2) → 비교군보다 높은 수준
• 공복혈당: 당신 105 | 비교군 평균 96 (±14.8) → 비교군보다 높은 수준
```


## 4.3 화면 플로우

```
[서비스 소개 화면]
  └── 서비스 목적, 데이터 출처, 주의문구, 시작 버튼
        │
        ▼
[입력 화면]
  ├── Step 1: 기본정보 (성별, 연령대, 키, 몸무게) ← 필수
  ├── Step 2: 건강검진 수치 (혈압, 혈당, 지질 수치 4종, 허리둘레) ← 선택
  ├── Step 3: 생활습관 (흡연, 음주, 운동) ← 권장
  ├── Step 4: 거주지역 ← 선택
  └── [자가점검 결과 보기] 버튼
        │
        ▼
[결과 화면]
  ├── 카드 1: 요약 (건강관리 유형, 입력 완성도)
  ├── 카드 2: 기본 건강지표 (BMI, 체중 상태)
  ├── 카드 3: 지질 수치 카드 (LDL·HDL·TG·총콜레스테롤 중 입력된 항목)
  ├── 카드 4: 주요 관리 포인트
  ├── 카드 5: 맞춤 생활관리 추천
  ├── 카드 6: 비교군 비교
  ├── 카드 7: 지역사회 건강관리 자원
  └── 카드 8: 주의문구 (필수)
```

## 4.4 금지 표현 목록

다음 표현은 코드·화면 어디서도 사용 금지:

| 금지 표현 | 대체 표현 |
|---------|----------|
| 고혈압입니다 | 혈압이 높은 범위에 해당할 수 있어 재측정과 상담을 권장합니다 |
| 당뇨입니다 | 공복혈당 수치가 높아 확인검사가 필요할 수 있습니다 |
| 치료가 필요합니다 | 의료기관 또는 보건소 상담을 권장합니다 |
| 약을 드셔야 합니다 | 약물 관련 판단은 의료진 상담이 필요합니다 |
| 반드시 좋아집니다 | 생활습관 관리가 도움이 될 수 있습니다 |
| 당신의 유형은 OO위험형입니다 | 입력값 기준, OO관리형에 가까운 것으로 나타났습니다 |
| 개선 가능성이 있습니다 | 관리 방향을 점검해볼 필요가 있습니다 |
